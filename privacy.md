# 网页内容高效处理 - 隐私政策

## 隐私保护承诺

"网页内容高效处理" 是一款注重用户隐私保护的浏览器扩展，我们承诺：

- 📱 **所有数据本地存储**：扩展不会将任何用户数据发送到远程服务器
- 🔒 **数据完全私密**：用户的所有操作和存储的内容仅保存在本地设备上
- 📝 **透明的数据处理**：本隐私政策详细说明扩展如何处理和存储数据

## 数据存储与使用

### 1. 存储的数据类型

扩展在本地存储以下类型的数据：

| 数据类型 | 存储位置 | 用途 |
|---------|---------|------|
| 用户设置 | chrome.storage.sync | 保存扩展的配置选项，如默认分类、图片提取设置等 |
| 收藏内容 | indexedDB (WebContentProcessor) | 存储用户收藏的网页内容、图片和相关信息 |
| 离线内容 | indexedDB (WebContentProcessorOffline) | 存储用户保存的离线内容和缓存的图片 |
| 提取的内容 | 临时内存 | 处理过程中的临时数据，不会持久存储 |

### 2. 数据处理方式

- **内容提取**：扩展仅在用户主动请求时提取网页内容，提取过程在本地完成
- **数据存储**：所有提取的内容和设置仅存储在用户本地设备上
- **数据使用**：存储的数据仅用于：
  - 在弹出页面中显示用户的收藏内容
  - 在选项页面中管理用户设置
  - 提供离线访问功能
  - 生成导出文件（如 Word、Markdown 等）

### 3. 数据共享

扩展**不会**：
- 与任何第三方共享用户数据
- 向远程服务器发送用户数据
- 收集用户的浏览历史或行为数据
- 追踪用户的使用情况或分析使用统计

## 权限说明

扩展请求以下权限，所有权限均用于本地功能实现，不会用于数据收集：

| 权限 | 用途 |
|------|------|
| storage | 存储用户设置和配置 |
| contextMenus | 添加右键菜单功能 |
| activeTab | 在当前活动标签页执行内容提取 |
| downloads | 允许用户下载生成的文件 |
| scripting | 在网页中注入脚本以实现内容提取 |
| <all_urls> | 允许扩展在任何网页上运行内容提取功能 |

## 数据管理

### 1. 数据导出

用户可以随时导出以下数据：
- **收藏内容**：导出为 JSON 格式文件
- **离线内容**：导出为 JSON 格式文件
- **设置**：包含在浏览器的同步存储中

### 2. 数据删除

用户可以：
- **删除单个收藏**：在弹出页面中删除特定的收藏内容
- **清空所有收藏**：通过选项页面批量删除所有收藏
- **删除离线内容**：在选项页面中删除特定的离线内容
- **清空所有离线内容**：在选项页面中批量删除所有离线内容
- **重置设置**：将所有设置恢复为默认值

### 3. 数据安全

- **本地存储**：所有数据存储在浏览器的本地存储中，受到浏览器安全机制的保护
- **数据隔离**：不同用户的数据完全隔离，不会相互访问
- **无网络传输**：由于数据不发送到网络，避免了网络传输中的安全风险
